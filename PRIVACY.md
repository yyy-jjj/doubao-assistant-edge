# 隐私政策 (Privacy Policy)

## 概述
本扩展程序（DouBao Assistant / 豆包对话目录助手）尊重并保护所有用户的个人隐私。本政策旨在说明本扩展程序如何处理用户数据。

## 数据收集与使用
- **不收集个人信息**：本扩展程序不会访问、收集、存储或传输任何可用于识别用户身份的个人信息（如姓名、邮箱、地址等）。
- **本地数据存储**：所有与对话目录相关的数据（如问题列表、搜索历史）仅存储在用户本地设备的浏览器存储中 (`chrome.storage.local`)。
- **无数据传输**：本扩展程序不会将任何数据发送到任何第三方服务器或服务。

## 数据安全
- 所有本地存储的数据仅对用户当前的浏览器会话可见，不会被其他网站或扩展程序访问。
- 用户可以随时通过浏览器扩展管理页面或清除浏览器数据来删除本扩展程序存储的所有信息。

## 第三方服务
本扩展程序不集成任何第三方分析、广告或数据收集服务。

## 政策更新
本隐私政策可能会根据扩展程序功能的更新而修订。更新后的政策将发布在本项目的 GitHub 仓库中。
